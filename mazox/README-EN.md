# YA NO GASTES DINERO EN EL MAZO

Programming language created in Python.

MAZO is a custom mini-language inspired by:
- Python
- Simple interpreters
- Custom languages
- The "YA NO GASTES DINERO EN EL MAZO" joke (among friends  meme)

---

# Features

▪︎ Custom command system  
▪︎ Variables  
▪︎ Custom colors  
▪︎ Multiple functions  
▪︎ <> based syntax  
▪︎ Compatible with Termux  
▪︎ Fully made in Python  

---

# Language philosophy

MAZO-X was created as a fun project to experiment with:
- parsers
- interpreters
- custom commands
- terminals
- ANSI colors
- syntax systems

All while keeping the meme:

"YA NO GASTES DINERO EN EL MAZO"

---

# Why does MAZO-X exist?

Everything started when me and my friend were playing Minecraft. Then he started saying: "Ya no gastes dinero en el mazo, mejor usa la lanza!" After that, I got annoyed because he kept repeating it every second. Eventually I got used to it and started joking along with him. It got to the point where I ended up creating this project: MAZO-X.

# Installation

## Requirements

- Python 3
- Termux or Linux

---

## Install Python in Termux

# 1. Install Python
pkg update
pkg install python

DO NOT RUN PYTHON YET!

# 2. Download the file

Download the file and extract it into the Download folder.

# 3. Run this command in Termux:
cd /storage/emulated/0/Download/mazox

# 4. Run this command in Termux:
python interprete.py

After this, you should already be inside MAZO-X.
