# YA NO GASTES DINERO EN EL MAZO

Lenguaje de programación creado en Python.

MAZO es un mini lenguaje personalizado inspirado en:
- Python
- Intérpretes simples
- Lenguajes personalizados
- Humor del "YA NO GASTES DINERO EN EL MAZO" (Meme entre amigos)

---

# Características

▪︎ Sistema de comandos propio  
▪︎ Variables  
▪︎ Colores personalizados  
▪︎ Funciones múltiples  
▪︎ Sintaxis basada en <>   
▪︎ Compatible con Termux    
▪︎ Hecho completamente en Python  

---

# Filosofía del lenguaje

MAZO-X fue creado como un proyecto divertido para experimentar con:
- parsers
- intérpretes
- comandos personalizados
- terminales
- colores ANSI
- sistemas de sintaxis

Todo mientras mantiene el meme de:

"YA NO GASTES DINERO EN EL MAZO"

---

# ¿Por qué existe MAZO-X?

Todo empezó cuando yo y mi amigo estabamos jugando minecraft. y luego el empezó a decir: "Ya no gastes dinero en el mazo, mejor usa la lanza!". después de eso no me gustaba que dijiera eso porque se me hacia muy repetitivo, ya que me lo decía cada segundo. despues de eso empecé a acostumbrarme hasta seguirle el juego. hasta llegué al punto de hacer esto, MAZO-X.

# Instalación

## Requisitos

- Python 3
- Termux o Linux

---

## Instalar Python en Termux

# 1. instala Python
pkg update

pkg install python

¡NO EJECUTES PYTHON TODAVIA!

# 2. Descarga el archivo

Descarga el archivo y extraelo en la carpeta Download.

# 3. Ejecuta este codigo en termux: 
cd /storage/emulated/0/Download/mazox

# 4 Ejecuta este codigo en termux:

python interprete.py

Despues de esto ya deberías estar en MAZO-X.
