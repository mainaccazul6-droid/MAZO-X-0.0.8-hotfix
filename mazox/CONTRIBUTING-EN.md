# How to contribute

You can help by:
- adding commands
- improving errors
- optimizing the parser
- creating examples
- improving documentation

---

# Rules

- Keep the MAZO style
- Use syntax with <>
- Do not break old commands
- Try to comment the code
- The syntax must include the letters from the phrase: ya no gastes dinero en el mazo
